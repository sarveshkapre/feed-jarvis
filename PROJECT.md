# PROJECT

## Requirements
- Node.js 20+

## Workflows
- Setup: `make setup`
- Dev: `make dev`
- Studio Dev: `npm run dev:web`
- Studio Smoke: `npm run smoke:web`
- Studio Browser E2E: `npm run e2e:web`
- Test: `make test`
- Lint: `make lint`
- Typecheck: `make typecheck`
- Build: `make build`
- Check (CI gate): `make check`
